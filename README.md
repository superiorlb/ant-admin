# Ant Admin

## 项目介绍
基于React18+Ant Design构建的后台模板，数据都是mock生成，仅供展示
### 项目界面

![alt](./src/assets//img/ant-admin.jpg)

## 准备工作

项目基于 pnpm + create-react-app 创建，需要安装 pnpm， 地址：https://www.pnpm.cn/

### 安装命令

```
npm install -g pnpm
```

### 安装依赖

```
pnpm install
```

## 启动命令

```
pnpm start
```

## 打包构建

```
pnpm build
```
